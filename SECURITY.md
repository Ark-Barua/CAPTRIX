# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.0.x | Yes |
| < 1.0.0 | No |

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Preferred process:
- Use GitHub private vulnerability reporting for this repository, if enabled.
- If private reporting is not available, contact the maintainers directly through repository owner channels.

When reporting, include:
- affected component/file
- reproduction steps
- impact assessment
- suggested mitigation (if known)

We will acknowledge receipt and provide updates as triage and fixes progress.
